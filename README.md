# One-Word PDF Reader

A simple web application that reads PDF files one word at a time, perfect for speed reading practice.

## Features

- Upload and parse PDF files
- Display text one word at a time
- Adjustable reading speed (60-900 words per minute)
- Dark/Light theme toggle
- Progress tracking

## How to Use

1. Open `index.html` in a web browser
2. Click "Choose PDF" to upload a PDF file
3. Adjust the speed slider if needed
4. Click "Start" to begin reading
5. Use "Reset" to start over

## Publishing

This is a static website that can be published to any static hosting service:

- **Netlify**: Drag and drop the folder to netlify.com
- **Vercel**: Import the folder to vercel.com
- **GitHub Pages**: Push to GitHub and enable Pages in repository settings
- **Cloudflare Pages**: Upload to pages.cloudflare.com

No build process required - just upload the files!
